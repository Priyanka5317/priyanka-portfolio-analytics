# Priyanka Mangrulkar | Data Analytics, Business Analysis, BI Portfolio

A role-targeted portfolio for Data Analytics, Business Analysis, and BI Development.
Single-file static site (HTML + Tailwind via CDN + Three.js) with a Data Aurora
chart-bar background and a cinematic intro. No build step.

## Deploy on Vercel
1. Import this repo at https://vercel.com/new
2. Framework preset: Other
3. Deploy.

## Structure
- `index.html` : the entire site
- `images/` : profile photo and project graphics
